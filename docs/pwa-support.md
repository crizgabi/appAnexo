# Suporte PWA no AppAnexo

Este backend fornece a base para o funcionamento do PWA, incluindo:
- Endpoints de autenticação (`/users/login`, `/users/refresh-token`);
- Suporte a cache de dados offline via API;
- Possibilidade futura de integração com notificações push (Web Push).

> Criado para fins de versinamento inicial e testes.
